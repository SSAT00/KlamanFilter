# KlamanFilter
Algorithm Kalman filter with graphics interface.

# what need to start
    * it is necessary to place the files "fond438.bin" and "EOP.DAT" in the directories "nm/fond/" and "nm/eop/" respectively.
    * install QtCreator "qt-opensource-windows-x86-5.14.2.exe"
    * install lib "boost 1.90.0" in directory "boost_1_90_0"
    * install lib "Eigen" in directory "toolboxEigen"
